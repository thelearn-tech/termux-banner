# termux-restore

If you messed up with bash.bashrc file
and something goes wrong 
with this script 
you can restore 
bash.bashrc file to original one

# Installation

*git clone https://github.com/thelearn-tech/termux-restore*

*cd termux-restore*

*chmod +x restore.sh*

*./restore.sh*
